# Life Character Sheet

An interactive v0.1 prototype built from Christopher's `life_character_sheet_prd.md`.

The first iteration is a static browser artifact with embedded CSS and JavaScript. It models the current life build as an RPG-style character sheet with:

- identity card, traits, and status effects;
- adjustable polarity sliders;
- derived resources;
- scenario presets;
- scenario simulator controls;
- local deterministic narrative projection;
- tradeoff notes;
- save/load/reset via `localStorage`;
- JSON and Markdown export.

This is not a productivity dashboard or a scientific assessment. It is a reflective life-design simulator: a mirror with sliders.
