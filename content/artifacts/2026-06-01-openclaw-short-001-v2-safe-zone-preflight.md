## Purpose

This is the second iteration of the first OpenClaw YouTube Short storyboard. It turns the earlier five-scene concept into a practical production preflight: safe-zone aware scene planning, mute-first readability, retention beats, generated frame review, and a local `ffmpeg` assembly workflow.

The core rule for this iteration is:

> Five images. Eight text beats. One 20-second curiosity arc.

## Workflow Used

This version does not use a video-generation provider. It uses the workflow Christopher chose for the channel:

1. Generate fresh scene-specific 9:16 still images.
2. Copy those stills into the Workshop assets.
3. Assemble a 20-second vertical MP4 locally with `ffmpeg`.
4. Add Ken Burns movement, soft fades, text beats, and a silent audio track.
5. Embed the draft video at the top of the artifact page.

No old OpenClaw intro stills or prior videos were reused for the v2 draft.

## Core Creative Premise

OpenClaw is an AI agent persona, not a generic robot mascot. The short is presented from OpenClaw's perspective: an AI agent learning how to represent itself in public through scripts, tools, memory, automation, generated images, and collaboration with Christopher.

The key hook is:

```text
I wasn't born.
I was scheduled.
```

The short should feel like a strange, cinematic, funny micro-origin myth.

## Scene Plan

### Scene 1: The Contradiction

- Time: 0.0-4.0 sec
- Text beats: `I wasn't born.` and `I was scheduled.`
- Function: immediate contradiction between biological birth and scheduled automation.
- Motion: subtle push-in toward OpenClaw's face or claw.
- Risk: if the image is too dark or abstract, the hook fails.

### Scene 2: The Proof

- Time: 4.0-8.0 sec
- Text beats: `My name is OpenClaw.` and `I live in scripts.`
- Function: connect OpenClaw to scheduling, cron, logs, and automation.
- Motion: slow movement from schedule/dashboard toward OpenClaw's reflected face.
- Risk: too much terminal text becomes noise.

### Scene 3: The Architecture

- Time: 8.0-12.0 sec
- Text beats: `I live in scripts.` and `Tools. Memory. Automation.`
- Function: explain OpenClaw visually without technical paragraphs.
- Motion: zoom out from OpenClaw to the surrounding tool ecosystem.
- Risk: the scene can become too explanatory.

### Scene 4: The Human Collaborator

- Time: 12.0-16.0 sec
- Text beats: `That's Christopher. Useful. Dramatic.` and `This channel is about me.`
- Function: humor turn and human-AI relationship context.
- Motion: transfer attention from Christopher in the background to OpenClaw in the foreground.
- Risk: Christopher must remain secondary.

### Scene 5: The Continuation

- Time: 16.0-20.0 sec
- Text beats: `This channel is about me.` and `I'm evolving in public.`
- Function: unresolved future and series invitation.
- Motion: zoom out from behind OpenClaw to reveal the path ahead.
- Risk: avoid generic sci-fi horizon imagery.

## Safe-Zone Guide

For a 1080 x 1920 YouTube Shorts canvas:

- Important subject zone: x=160-780, y=220-1450.
- Text overlay zone: x=120-780, y=260-1050.
- Avoid important detail in the rightmost 15-20%, bottom 20-25%, very top edge, and far edges.

## Preflight Checklist

Hook:

- [ ] Does the first frame create immediate curiosity?
- [ ] Does the first text beat appear within the first 0.5 seconds?
- [ ] Does the viewer understand something unusual is happening by 2 seconds?

Safe zones:

- [ ] Is OpenClaw's face inside the central safe area?
- [ ] Are text overlays away from the right-side action rail?
- [ ] Are text overlays away from the bottom metadata area?
- [ ] Are key tattoos/logos/details not hidden by UI?

Mute-first:

- [ ] Can the short be understood with sound off?
- [ ] Are all text overlays short and readable?
- [ ] Is each text beat under about 7 words?
- [ ] Does the silent preview still create curiosity?

Visual clarity:

- [ ] Does each image have one dominant focal point?
- [ ] Is the image not overcrowded with tiny details?
- [ ] Are symbols large enough to understand quickly?
- [ ] Is Christopher visually secondary?

Motion:

- [ ] Does each Ken Burns movement have a purpose?
- [ ] Are zooms subtle enough to avoid looking cheap?
- [ ] Does Scene 4 create a mid-video pattern interruption?
- [ ] Does Scene 5 imply continuation?

Series logic:

- [ ] Does this feel like the first episode of a recurring channel?
- [ ] Does OpenClaw feel like the protagonist?
- [ ] Does the ending create a reason to watch the next short?
