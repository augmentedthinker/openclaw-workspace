Friendly Robot T-Shirt Design Test

Created 2026-05-15 · late morning EDT Brand context: Augmented Thinker / Fourthwall-style t-shirt store exploration Deliverables: transparent SVG source + 4500×5400 transparent PNG Status: first workflow proof, not final product art
The old workflow was generate, upscale, remove background, clean up, export. This test collapses that into a vector-first design with transparency from the beginning.

Christopher asked whether OpenClaw can now handle the old tedious t-shirt pipeline: create an AI-style design, upscale it, remove the background, and make it usable for a storefront like Fourthwall.

This first test takes the cleaner route: instead of generating a raster image and then fighting to remove the background, the design was built as a vector/SVG graphic first. That means the background is transparent by construction, the edges stay crisp, and the design can be exported at print size without a separate upscaling step.

What was produced

  • Editable vector source: friendly-robot-face-tshirt.svg
  • Print-size transparent PNG: friendly-robot-face-tshirt-transparent-4500x5400.png
  • Canvas: 4500×5400 pixels, a common large merch-art size.
  • Background: transparent, not removed after the fact.
  • Style: friendly robot face, cyan/amber Augmented Thinker palette, no text, simple screen-print-friendly shapes.

What this proves

For simple graphic designs, OpenClaw can now avoid much of the old multi-app pipeline:

  1. Design directly in SVG or generate a raster concept.
  2. Keep transparency from the start whenever possible.
  3. Export a print-size PNG for storefront upload.
  4. Host the design in the Workshop for review from phone or laptop.

Current limitations

This is a strong workflow proof, but not yet a finished clothing line. The design should still be checked against Fourthwall’s exact upload requirements, print-safe margins, and shirt-color choices. It may need a white-outline variant for black shirts, a simplified one-color version for cheaper printing, or a more distinctive Augmented Thinker brand mark.

The important part: the process is now much more controllable. If the art can be expressed as vector shapes, background removal and upscaling become mostly unnecessary.
